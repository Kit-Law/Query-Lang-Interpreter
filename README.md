# Query-Lang-Interpreter

This is a domain specific programming language for querying CSV documents, called CQL.

## Compilation

Use stack or cabal to compile:

```cmd
stack build --copy-bins --local-bin-path .
```

## Usage

```cmd
.\csvql [script_file].cql
```
